ATM Simulator System

ATM Simulator System is a mini project developed using html, css and javaScript. This system simulates real ATM banking operations in a simple and interactive way.

  • Cash Withdrawal ̶ – Withdraw available cash from account
  • Check Balance̶ – View current account balance
  • Change PIN ̶ – Securely change ATM PIN
  • Cash Deposit ̶ – Deposit cash into account
  • Mini Statement ̶ – View recent transaction history
  • Exit ̶ – Safely exit the ATM system
  • No Cash̶ – Displays message when cash withdraw
