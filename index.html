ATM Simulator System Project

ATM Simulator System Project is a mini web application developed using HTML, CSS and JavaScript. This ATM Simulator System project simulates real banking operations such as cash withdrawal, deposit, balance enquiry, mini statement and PIN change.

  • Cash Withdrawal – Withdraw available cash from account
  • Check Balance – View current account balance
  • Change PIN – Securely change ATM PIN
  • Cash Deposit – Deposit cash into account
  • Mini Statement – View recent transaction history
  • Exit – Safely exit the ATM system
  • No Cash – Displays message when cash withdraw
